# chuahpstix-league-viz

Static web app for **chuahpstix** with a deeper Riot-backed League page plus a smaller TFT appendix.

## What it shows now

### League

- full currently fetchable Riot-visible match-v5 history window, paginated in 100-match chunks
- recent-10 snapshot plus deep-window record/context
- champion identity tracker that combines Riot champion mastery with actual match outcomes
- signature play callouts based only on metrics the match payload actually exposes
- playtime personality splits by time band in America/New_York
- worst-enemy correlation table with sample-size caution
- personal records pulled from the same fetched history window
- teammate leaderboard from the same deep sample

### TFT

- recent public-profile snapshot from tactics.tools

## Local preview

```bash
npm run build
open index.html
```

The page reads from `data/chuah-data.json`, generated by `scripts/generate-data.mjs`.

## Data sources

- **League:** Riot account-v1 + summoner-v4 + champion-mastery-v4 + match-v5
- **TFT:** tactics.tools public profile payload parsed from `__NEXT_DATA__`

The TFT fallback still exists because this environment is accepting League endpoints but not giving reliable Riot TFT match history access.
