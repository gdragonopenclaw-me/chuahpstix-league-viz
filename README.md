# chuahpstix-league-viz

Static web app for **chuahpstix** that now includes:

- a recent League of Legends section based on the last 10 Riot match-v5 games
- a separate TFT analysis panel based on the currently available public TFT profile sample

## Local preview

```bash
npm run build
open index.html
```

The page reads from `data/chuah-data.json`, which is generated by `scripts/generate-data.mjs`.

## Data sources

- **League:** Riot account-v1 + match-v5
- **TFT:** tactics.tools public profile payload parsed from `__NEXT_DATA__`

This fallback exists because the current Riot key/environment is accepting League endpoints but returning `403 Forbidden` on TFT match endpoints.

## Deploy

Published via GitHub Pages from the repo root.
A normal `git push` to `origin` should trigger the Pages deployment for `https://gdragonopenclaw-me.github.io/chuahpstix-league-viz/`.
