# chuahpstix-league-viz

Static GitHub Pages app for a small precomputed League friend-cluster dashboard.

## What it shows now

### Dashboard v2

- multi-player Riot-backed payload with client-side switching between a tiny explicit allowlist
- clickable teammate links in the most-played teammates table
- deep-sample snapshot, recent-10 strip, queue mix, and teammate leaderboard per player
- premade synergy buckets for exactly 0, 1, 2, 3, or 4 top-10 teammates on the same team
- player archetype spider/radar chart built from within-lobby percentile signals
- champion identity tracker using Riot mastery plus Data Dragon class/subclass tags
- comfort-vs-success class reads by champion tag bucket
- signature play, playtime personality, worst-enemy reads, and personal records
- teammate network graph for the root player plus top 10 teammates, including teammate-to-teammate edges

### Shareable quiz

- `quiz.html` reads the same generated payload
- supports the selected player through `?player=<riot-id>`
- randomized question selection from actual stats: champ identity, teammates, queue habits, playtime, records, and signature signals

## Local preview

```bash
npm run build
python3 -m http.server 8008
open http://127.0.0.1:8008/index.html
```

The page reads from `data/chuah-data.json`, generated by `scripts/generate-data.mjs`.

## Data sources

- **League:** Riot account-v1 + summoner-v4 + champion-mastery-v4 + match-v5
- **Champion metadata:** Riot Data Dragon champion tags
